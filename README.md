# Linux avanzado para DevOps

Toolkit avanzado de administracion Linux para operaciones y DevOps.

---

## Descripcion

Proyecto avanzado que cubre administracion del sistema, seguridad, monitoreo y redes.

### Que hace este proyecto?

- **Servicios**: systemctl, journalctl, service
- **Automatizacion**: crontab y bash scripting
- **Seguridad**: ufw, iptables, fail2ban
- **Monitoreo**: htop, iotop, vmstat
- **Logs**: /var/log y tail -f
- **Redes avanzadas**: ss, tcpdump, nmap

---

## Caracteristicas principales

| Caracteristica | Descripcion |
|----------------|-------------|
| **Portal avanzado** | Secciones por dominio DevOps |

## Stack tecnologico

- **Frontend**: HTML5, CSS3 moderno, JavaScript Vanilla
- **Scripts**: Bash

## Instalacion y uso

### Requisitos

- Navegador moderno
- Linux o WSL para scripts bash

### Uso

```bash
# Abrir el sitio
start index.html

# Ejecutar scripts
bash bash/server_monitor.sh
bash bash/backup.sh
bash bash/log_watch.sh
```
